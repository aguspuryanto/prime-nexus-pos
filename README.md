# PRIME NEXUS POS

PRIME NEXUS POS telah berhasil diimplementasikan sebagai solusi kasir modern yang ringan dan responsif untuk UMKM.

## Fitur Utama yang Telah Diimplementasikan:

### Desain Cross-Platform & Responsif:
- **Desktop**: Tampilan profesional dengan sidebar navigasi dan keranjang belanja permanen di sisi kanan.
- **Mobile**: Desain yang dioptimalkan untuk layar sentuh dengan Floating Cart (keranjang melayang) untuk akses cepat saat memesan.

### Manajemen Kasir & Keamanan:
- Sistem login berbasis PIN 4 digit untuk keamanan akses kasir.
- Manajemen peran (Admin/Kasir) untuk pembatasan akses fitur sensitif.

### Sistem Transaksi Lengkap:
- **Metode Pembayaran**: Mendukung Tunai (dengan kalkulator kembalian otomatis), QRIS, dan Kartu Debit.
- **Simpan Tagihan (Open Table)**: Fitur untuk menahan pesanan pelanggan (meja) sebelum pembayaran dilakukan.
- **Riwayat Transaksi**: Laporan lengkap semua transaksi yang telah dilakukan.

### Manajemen Menu:
- Kategorisasi produk (Makanan, Minuman, Snack).
- Fitur pencarian menu yang cepat dan responsif.

### Dukungan Printer Thermal:
- Antarmuka simulasi status printer yang siap mencetak struk dalam format Standar atau Klasik.

### Pengalaman Pengguna (UX):
- **Dark Mode**: Mode gelap untuk kenyamanan penggunaan di lingkungan minim cahaya.
- **Animasi Halus**: Menggunakan motion untuk transisi antar menu dan interaksi keranjang yang elegan.

## Detail Teknis:
- **Frontend**: React 19, Tailwind CSS 4, Lucide Icons, Motion.
- **Backend**: Express.js (Node.js).
- **Database**: SQLite (better-sqlite3) untuk penyimpanan data lokal yang cepat dan andal.

Aplikasi ini siap digunakan untuk meningkatkan efisiensi operasional bisnis Anda!
